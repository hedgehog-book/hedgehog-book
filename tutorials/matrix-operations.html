Basic Matrix Operations

There are a few basic operations that's are most necessary in linear algebra.

Matrix addition A+BA+BA+B

For an m×nm \times nm×n matrix AAA and an m×nm \times nm×n matrix BBB, the matrix addition C=ABC=ABC=AB is defined as an m×nm \times nm×n matrix such that

For an m×nm \times nm×n matrix AAA, the matrix transpose ATA^{T}A​T​​ is defined as an n×mn \times mn×m matrix such that

Scalar multiplication cAcAcA

For an m×nm \times nm×n matrix AAA and a scalar number ccc, the scalar multiplication X=cAX=cAX=cA is defined as an m×nm \times nm×n matrix such that

Matrix Inverse A−1A^{-1}A​−1​​

For an n×nn \times nn×n square matrix AAA, the inverse of matrix A−1A^{-1}A​−1​​ is defined as an n×nn \times nn×n matrix such that
